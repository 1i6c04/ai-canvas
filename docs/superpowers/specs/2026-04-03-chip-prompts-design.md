# Design: 推薦提示 Chip + 骰子功能

**Date:** 2026-04-03  
**Status:** Approved

## 問題背景

AI Canvas 分享到 Threads 後，訪客進站率低，且進站後幾乎沒有人輸入提示。根因分析：

1. **傳播層**：Threads 貼文無視覺內容（純文字 + 留言放連結），演算法不推廣，觸及 ~10 人
2. **轉化層**：訪客不知道要輸入什麼，空白輸入框缺乏引導

本 spec 解決轉化層問題，搭配 Threads 貼文腳本同步提升傳播。

---

## 功能設計：推薦提示 Chip + 骰子

### 外觀佈局

控制面板（Shadow DOM）內，輸入框上方新增一排 chip 標籤與骰子按鈕：

```
[⭐ 星空背景] [🌈 彩虹標題] [💥 霓虹卡片] [🌊 海浪動畫]  🎲
──────────────────────────────────────────────────────────
[輸入你的指令...                                    ] [送出]
```

- 預設顯示 4 個 chip
- 🎲 按鈕靠右對齊

### 互動行為

| 操作 | 行為 |
|------|------|
| 點擊 chip | 將 chip 文字填入輸入框，自動 focus 輸入框，**不**自動送出（讓用戶可修改） |
| 點擊 🎲 | 從 pool 隨機抽取 4 個新 chip，排除當前已顯示的，更新畫面 |

### Chip Pool（約 14 個）

```
星空背景、彩虹標題文字、霓虹燈卡片、海浪動畫效果、
把背景變成熔岩、讓按鈕閃爍、文字變成金色、
卡片加上玻璃質感、標題加上打字機效果、
背景變成極光、讓整個頁面旋轉360度、
卡片變成浮空效果、文字加上彩色陰影、背景變成像素風格
```

### 樣式規範

- Chip 樣式定義在 Shadow DOM 內部 `<style>` 中，**不受** AI 生成 CSS 影響
- Chip 外觀：圓角膠囊、淺色背景、hover 時加深，與現有控制面板風格一致
- 🎲 按鈕：純 icon，無邊框，hover 有旋轉動畫

## 實作範圍

### 修改檔案
- `index.php`：唯一修改目標
  - 在 Shadow DOM 控制面板的 HTML 結構中新增 chip 列
  - 在 Shadow DOM 內部 `<style>` 新增 chip / 骰子樣式
  - 在 inline JS 新增 chip pool 陣列、renderChips()、handleDice() 函式

### 不修改
- `api.php`、`db.php`、`config.php`、`style.css` — 純前端功能，不需後端變動

## Threads 貼文腳本

### 錄影流程（30 秒）
1. 開啟網頁，展示目前累積的視覺效果（3 秒）
2. 點一個 chip（例如「星空背景」），畫面即時變化（5 秒）
3. 點 🎲 骰子，chip 換新組合（3 秒）
4. 點另一個 chip（例如「霓虹卡片」），再次變化（5 秒）
5. 拉遠鏡頭，顯示整個畫面的累積效果（3 秒）

### 發文文案

**主文（不含連結）：**
```
做了一個任何人都可以改造的網頁

你輸入指令，AI 就會幫你改變頁面外觀
所有人的改動會永久累積在同一個頁面上

我輸入「星空背景」它就變成這樣 👇
[附錄影]
```

**留言（第一則）：**
```
連結在這裡，歡迎進來改造 → [URL]
```
